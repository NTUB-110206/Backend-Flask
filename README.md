# backend-flask

### env
#### You must fill the following variables in your app/setting.py file:
- SECRET_KEY
- SQLALCHEMY_DATABASE_URI

### requirements package
- for Windows
```
pip install -r requirements.txt
```
- for macOS
```
pip3 install -r requirements.txt
```

### Start
- for Windows
```
py -3 runserver.py
python runserver.py
```
- for macOS
```
python3 runserver.py
```
